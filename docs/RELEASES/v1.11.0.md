### Operational notes

- **No deploy required** for the documentation cleanup itself â€”
  it's pure prose. The runtime brand rename (icons, domains, DB
  models, code paths) was deployed in earlier PRs and is already

- **Rollback** for the rename wave as a whole is non-trivial: the
  Prisma migration in PR #26 was destructive (rename + drop old

  brand-related regression appears, the path is forward-fix, not
  backward-revert.
